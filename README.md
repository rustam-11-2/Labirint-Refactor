# Путаница
## Игра «Лабиринт из ниток».

**Целевая аудитория:** дети младшего школьного возраста (обучающая/развивающая/логическая игра).

**Цель игры:**

	Игрок должен правильно сопоставить буквы из первой строки таблицы с ячейками последней строки, ориентируясь на визуальный лабиринт из линий («ниток»). 
	При правильном сопоставлении нижняя ячейка заполняется буквой и воспроизводится хвалебный звук. 
	После заполнения всех нижних ячеек отображается итоговое слово и несколько картинок в левой части экрана. 
	Игрок должен выбрать картинку, на которой изображено значение этого слова. Игрок выигрывает, если он выбрал правильную картинку. 

**Техническое задание:**

	Возможность выбрать любой из представленных уровней (1 уровень, 2 уровень и тд.) в отдельном окне.

	Отображение выбранного уровня. Тоже в отдельном окне.

	Реализовать проигрывание звуков (звуки похвалы/огорчения) при сопоставлении ячеек из первой и последней строк.

	Отображение сообщения об успешном завершении уровня. 

	Отображение полученного слова, состоящего из собранных букв. 

**Игровая механика:**

_Логика хода:_

	Игрок кликает по букве в верхней строке.
	
	Выбранная буква визуально подсвечивается.
	
	Игрок кликает по одной из ячеек в нижней строке.
	
	Выполняется проверка соответствия: Проверяется, соединена ли выбранная буква с выбранной нижней ячейкой по «нитке».

_Успешный выбор:_

	Нижняя ячейка заполняется выбранной буквой.

	Проигрывается хвалебный звук.

_Ошибочный выбор:_
	
	Ячейка не заполняется.
	
	Проигрывается звук ошибки.
	
	Выбор сбрасывается.

_Завершение игры:_
	
	После заполнения всех нижних ячеек появляются картинки.
	
	Игра завершается, если игрок выбрал правильную картинку.
