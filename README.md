# 1. Название проекта
Компьютерная игра «Путаница» для изучения осетинского языка.
# 2. Короткое описание назначения проекта
Компьютерная игра для изучения осетинского языка детьми через ассоциацию изображений.

**Цель игры:**

Игрок должен правильно сопоставить буквы из первой строки таблицы с ячейками последней строки, ориентируясь на визуальный лабиринт из линий («ниток»). 
При правильном сопоставлении нижняя ячейка заполняется буквой и воспроизводится хвалебный звук. 

После заполнения всех нижних ячеек отображается итоговое слово и несколько картинок в левой части экрана. 
Игрок должен выбрать картинку, на которой изображено значение этого слова. Игрок выигрывает, если он выбрал правильную картинку. 
# 3. Функциональные возможности
* **Выбор уровня:** На форме выбора уровня игрок выбирает любой уровень и открывается окно с самой игрой.
* **Проигрывание звука:** Проигрывание звука при сопоставлении 2 ячеек игрового поля или выбора картинки, на которой изображено значение слова. 
* **Сохранение и чтение прогресса:** При открытии формы выбора уровня считываются пройденные уровни. После прохождения уровня этот уровень сохраняется в файл для дальнейшего считывания. 
* **Создание уровня:** Возможность создание нового уровня.
# 4. Руководство администратора
* Операционная система: Windows 10 / 11.
* Платформа: .NET Framework 4.7.2.+
* Файловая структура: В папке с исполняемым .exe файлом должны находиться:
  * Папка `Sounds/`с аудиофайлами `.wav`.
  * Папка `GameLvls/` где хранятся уровни в виде `.txt` файлов и файл `progress.txt` .
  * Папка `Images/` где хранятся картинки букв осетинского алфавита.
  * Папка `ImagesMeaningsOfWords/` где хранятся картинки с изображением значения загаданного уровня.
  * Папка `ImageThreadsTypeOne/` где хранятся картинки лабиринта.
# 5. Руководство пользователя
1. Запустите файл `Labirint.exe`.
2. Нажмите кнопку `НАЧАТЬ`.
3. В открывшемся окно выберите один из уровней.
4. На игровом поле сопоставить буквы из первой строки таблицы с ячейками последней строки, ориентируясь на визуальный лабиринт из линий.
5. После заполнения всех нижних ячеек отображается итоговое слово и несколько картинок в левой части экрана.
6. Выберите одну из картинок, на которой изображено значение этого слова.
# 6. Руководство программиста
Приложение реализовано на языке C# (Windows Forms). Проект разделен на классы и слой графического интерфейса:
### Классы проекта:
#### 1. Класс `SoundManager`
* **Поля:**
  * `completedLevel` (`SoundPlayer`) - переменная для воспроизведения звука завершения уровня.
  * `corChoice` (`SoundPlayer`) - переменная для воспроизведения звука правильного выбора.
  * `wrongChoice` (`SoundPlayer`) - переменная для воспроизведения звука неправильного выбора.
* **Методы:**
  * `PlayCorrect()` - воспроизводит звук при правильном выборе ячеек пользователем.
  * `PlayWrong()` - воспроизводит звук при неправильном выборе пользователя.
  * `PlayCompletedLevel()` - воспроизводит звук при правильном выборе изображения значения слова и при (завершение уровня).

#### 2. Класс `Cell`
* **Свойства:**
  * `enum CellType` - перечесление возможных типов клетки (Letter, Thread, EmptyCell).
  * `X` (`int`) - коррдината X клетки.
  * `Y` (`int`) - коррдината Y клетки
  * `Letter` (`string`) - значение карточки.
  * `Type` (`CellType`) - тип клетки.
  
#### 3. Класс `Level`
* **Свойства:**
  * `LvlRandomWord` (`string[]`, `get`, `set`) - массива символов "слово уровня" в перемешенном виде.
  * `LvlWord` (`string[]`, `get`, `set`) - массив символов "слово уровня".
  * `PictureBoxCorrextIndex`(`int`, `get`, `set`) - индекс "правильного" pictureBox.
  * `PictureBoxImagesName` (`string[]`, `get`, `set`) - названия картинок, которые будут загружаться в pictureBox'ы из папки где хранятся картинки.
  * `LevelType` (`int`, `get`, `private set`) - тип уровня. Зависит от длины слова.
* **Методы:**
  * `LoadLevel(`int lvlName`)` - инициализирует свойства класса.
  * `GetLevelType(`string[] word`)` - возвращает тип уровня.
  * `SaveProgress(`string lvl`)` - сохраняет прогресс.

  #### 4. Класс `GameField`
  * **Поля и Свойства:**
    * `GAMEMATRIXROWCOUNT` (`int`, `get`) - количество строк; всегда возвращает 4.
    * `gameMatrixColumnCount` (`int`) - количество столбцов.
    * `Cells` (`Cell[]`) - матрица игрового поля, состоящая и клеток типа Cell.
    * `Level` (`Level`, `get`, `private set`) - хранение параметров текущего уровня.
  * **Методы:**
    * `FixMatrix()` - присвоение координат каждому элементу матрицы.
    * `CreateMatrix()` - задает каждому элементу значение: символ и тип клетки.
    * `CheckAnswer(`Cell selectedLetter`, `Cell answer`)` - проверяет совпадают ли значение 2 выбранных ячеек.
    * `AllLetterWasFind()` - проверяет сопоставлены ли все ячейки.

  #### Слой графического интерфейса (Windows Forms):
  * `GameMainMenu` - начальное окно. Главное меню игры. 
  * `NewFormSelectLvl` - окно выбора выбора уровня. Считывается прогресс игрока. Пройденные уровни выделяются зеленым цветом.
  * `FormLabirint` - основное окно. Содержится `DataGridView`, несколько `PictureBox` и кнопку `Назад`. Отображение матрицы в `DataGridView`. Оброботка нажатий пользователя в методе dataGridViewLabirint_CellClick.
  * `CreateNewLevelForm` - окно, которое отвечает за создание нового уровня.
